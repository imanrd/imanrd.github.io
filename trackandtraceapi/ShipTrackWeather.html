post(request: HttpRequest) → HttpResponse[source]

Handles POST requests to retrieve weather and shipment data.

Args: - request: HTTP request object.

Returns: - HttpResponse: Rendered HTML template with weather and shipment data.

This method processes POST requests to retrieve weather and shipment data based on the input provided. It expects a payload containing ‘tracking_number’ and ‘carrier’ fields. If the request is made from the Swagger UI, it allows default values (‘TN12345680’ for tracking number and ‘DPD’ for the carrier) to be used if no input is provided. The method fetches shipment details and location information based on the provided tracking number and carrier, then fetches weather data for that location. Finally, it renders an HTML template with the retrieved weather and shipment data.
